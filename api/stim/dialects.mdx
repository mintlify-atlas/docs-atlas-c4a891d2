## Overview

Bloqade's STIM module organizes operations into four dialects:

- **Gate**: Quantum gates (Clifford gates, rotation gates, controlled gates)
- **Collapse**: Measurement and reset operations
- **Noise**: Error models and noise channels
- **Auxiliary**: Detectors, observables, and circuit metadata

All operations are available as Python functions in the `bloqade.stim` namespace.

## Auxiliary Dialect

Metadata, detectors, and observables for error correction.

### Measurement Records

#### rec

```python
def rec(id: int) -> RecordResult
```

Reference a measurement record by index.

<ParamField path="id" type="int" required>
  Record index. Negative indices reference recent measurements (-1 is the most recent).
</ParamField>

<ResponseField name="RecordResult" type="RecordResult">
  A reference to the measurement result that can be used in detectors and observables.
</ResponseField>

```python
stim.mz(p=0.0, targets=(0, 1))
rec0 = stim.rec(-2)  # Result of first measurement
rec1 = stim.rec(-1)  # Result of second measurement
```

### Pauli Strings

#### pauli_string

```python
def pauli_string(
    string: tuple[str, ...],
    flipped: tuple[bool, ...],
    targets: tuple[int, ...]
) -> PauliString
```

Create a Pauli string object for use in `spp()` and `mpp()`.

<ParamField path="string" type="tuple[str, ...]" required>
  Pauli operators ("X", "Y", "Z", or "I").
</ParamField>

<ParamField path="flipped" type="tuple[bool, ...]" required>
  Whether each Pauli operator is negated.
</ParamField>

<ParamField path="targets" type="tuple[int, ...]" required>
  Qubit indices for each Pauli operator.
</ParamField>

<ResponseField name="PauliString" type="PauliString">
  A Pauli string object that can be used in Pauli product operations.
</ResponseField>

```python

xyz = stim.pauli_string(
    string=("X", "Y", "Z"),
    flipped=(False, False, False),
    targets=(0, 1, 2)
)

# Use in measurement
stim.mpp(p=0.0, targets=(xyz,))

# Create -XXX string (negated)
neg_xxx = stim.pauli_string(
    string=("X", "X", "X"),
    flipped=(True, False, False),  # Negated
    targets=(0, 1, 2)
)
```

## Complete Example

Here's a complete example using operations from all dialects:

```python
from bloqade import stim

@stim.main
def error_correction_cycle():
    # Set qubit coordinates
    stim.qubit_coords(coord=(0, 0), target=0)
    stim.qubit_coords(coord=(1, 0), target=1)
    stim.qubit_coords(coord=(0, 1), target=2)
    
    # Initialize qubits
    stim.rz(targets=(0, 1, 2))
    
    # Create entanglement
    stim.h(targets=(0,))
    stim.cx(controls=(0,), targets=(1,))
    stim.cx(controls=(1,), targets=(2,))
    
    stim.tick()  # End of gate layer
    
    # Apply noise
    stim.depolarize1(p=0.001, targets=(0, 1, 2))
    
    # Measure stabilizers
    stim.mz(p=0.01, targets=(0, 1, 2))
    
    # Define detectors
    rec0 = stim.rec(-3)
    rec1 = stim.rec(-2)
    rec2 = stim.rec(-1)
    stim.detector(coord=(0, 0, 0), targets=(rec0, rec1))
    stim.detector(coord=(1, 0, 0), targets=(rec1, rec2))
    
    # Define logical observable
    stim.observable_include(idx=0, targets=(rec0, rec2))

# Compile and simulate
circuit = stim.Circuit(error_correction_cycle)
print(circuit)

# Sample detectors
detector_sampler = circuit.compile_detector_sampler()
samples = detector_sampler.sample(shots=1000)
```
