The Cirq integration provides bidirectional conversion between Bloqade kernels and Cirq circuits, enabling interoperability with the Cirq ecosystem.

## Circuit Emission

### emit_circuit

```python
from bloqade.cirq_utils import emit_circuit

circuit = emit_circuit(
    mt,
    circuit_qubits=None,
    args=(),
    ignore_returns=False
)
```

Converts a squin.kernel method to a cirq.Circuit object.

<ParamField path="mt" type="ir.Method" required>
  The kernel method from which to construct the circuit.
</ParamField>

<ParamField path="circuit_qubits" type="Sequence[cirq.Qid] | None" default="None">
  A list of qubits to use as the qubits in the circuit. If None, `cirq.LineQubit`s are inserted for every `squin.qalloc` statement in the order they appear inside the kernel.
  
  **Note**: If a list of qubits is provided, ensure there are sufficient qubits for the resulting circuit.
</ParamField>

<ParamField path="args" type="tuple" default="()">
  The arguments of the kernel function from which to emit a circuit.
</ParamField>

<ParamField path="ignore_returns" type="bool" default="False">
  If False, emitting a circuit from a kernel that returns a value will error. Set to True to ignore return values.
</ParamField>

<ResponseField name="return" type="cirq.Circuit">
  The generated Cirq circuit.
</ResponseField>

#### Example: Basic Usage

```python
from bloqade import squin
from bloqade.cirq_utils import emit_circuit

@squin.kernel
def main():
    q = squin.qalloc(2)
    squin.h(q[0])
    squin.cx(q[0], q[1])

circuit = emit_circuit(main)
print(circuit)
```

#### Example: Custom Qubits

```python
from bloqade import squin
from bloqade.cirq_utils import emit_circuit
import cirq

@squin.kernel
def main():
    q = squin.qalloc(2)
    q2 = squin.qalloc(3)
    squin.cx(q[1], q2[2])

# Custom list of qubits on grid
qubits = [cirq.GridQubit(i, i+1) for i in range(5)]

circuit = emit_circuit(main, circuit_qubits=qubits)
print(circuit)
```

#### Example: Composing Kernels

```python
from bloqade import squin
from bloqade.cirq_utils import emit_circuit
from kirin.dialects import ilist
from typing import Literal

@squin.kernel
def entangle(q: ilist.IList[squin.qubit.Qubit, Literal[2]]):
    squin.h(q[0])
    squin.cx(q[0], q[1])

@squin.kernel
def main():
    q = squin.qalloc(2)
    entangle(q)

circuit = emit_circuit(main)
print(circuit)
```

## Circuit Loading

### load_circuit

```python
from bloqade.cirq_utils import load_circuit

kernel = load_circuit(
    circuit,
    kernel_name="main",
    dialects=kernel,
    register_as_argument=False,
    return_register=False,
    register_argument_name="q",
    globals=None,
    file=None,
    lineno_offset=0,
    col_offset=0,
    compactify=True
)
```

Converts a cirq.Circuit object into a squin kernel.

<ParamField path="circuit" type="cirq.Circuit" required>
  The circuit to load.
</ParamField>

<ParamField path="kernel_name" type="str" default="main">
  The name of the kernel to load.
</ParamField>

<ParamField path="dialects" type="ir.DialectGroup" default="squin.kernel">
  The dialects to use.
</ParamField>

<ParamField path="register_as_argument" type="bool" default="False">
  Determine whether the resulting kernel function should accept a single `ilist.IList[Qubit, Any]` argument that is a list of qubits used within the function. This allows you to compose kernel functions generated from circuits.
</ParamField>

<ParamField path="return_register" type="bool" default="False">
  Determine whether the resulting kernel function returns a single value of type `ilist.IList[Qubit, Any]` that is the list of qubits used in the kernel function. Useful when you want to compose multiple kernel functions generated from circuits.
</ParamField>

<ParamField path="register_argument_name" type="str" default="q">
  The name of the argument that represents the qubit register. Only used when `register_as_argument=True`.
</ParamField>

<ParamField path="globals" type="dict[str, Any] | None" default="None">
  The global variables to use.
</ParamField>

<ParamField path="file" type="str | None" default="None">
  The file name for error reporting.
</ParamField>

<ParamField path="lineno_offset" type="int" default="0">
  The line number offset for error reporting.
</ParamField>

<ParamField path="col_offset" type="int" default="0">
  The column number offset for error reporting.
</ParamField>

<ParamField path="compactify" type="bool" default="True">
  Whether to compactify the output.
</ParamField>

<ResponseField name="return" type="ir.Method">
  The generated squin kernel method.
</ResponseField>

#### Example: Basic Loading

```python
import cirq
from bloqade.cirq_utils import load_circuit

# Pick a qubit
qubit = cirq.GridQubit(0, 0)

# Create a circuit
circuit = cirq.Circuit(
    cirq.X(qubit)**0.5,  # Square root of NOT
    cirq.measure(qubit, key='m')  # Measurement
)

# Load the circuit as squin
main = load_circuit(circuit)

# Print the resulting IR
main.print()
```

#### Example: Composing Circuits

```python
import cirq
from bloqade.cirq_utils import load_circuit
from bloqade import squin

q = cirq.LineQubit.range(2)
circuit = cirq.Circuit(cirq.H(q[0]), cirq.CX(*q))

get_entangled_qubits = load_circuit(
    circuit, return_register=True, kernel_name="get_entangled_qubits"
)

entangle_qubits = load_circuit(
    circuit, register_as_argument=True, kernel_name="entangle_qubits"
)

@squin.kernel
def main():
    qreg = get_entangled_qubits()
    qreg2 = squin.qalloc(1)
    entangle_qubits([qreg[1], qreg2[0]])
    return squin.qubit.measure(qreg2)
```

## Circuit Parallelization

Utilities for optimizing circuit parallelization using linear programming.

### parallelize

```python
from bloqade.cirq_utils import parallelize

optimized_circuit = parallelize(
    circuit,
    hyperparameters=None,
    auto_tag=True
)
```

Use linear programming to reorder a circuit for optimal parallel execution.

<ParamField path="circuit" type="cirq.Circuit" required>
  The static circuit to be optimized.
</ParamField>

<ParamField path="hyperparameters" type="dict[str, float] | None" default="None">
  Hyperparameters for the optimization:
  - `"linear"`: float (default: 0.01) - The linear cost of each gate
  - `"1q"`: float (default: 1.0) - The quadratic cost of 1q gates
  - `"2q"`: float (default: 2.0) - The quadratic cost of 2q gates
  - `"tags"`: float (default: 0.5) - The default weight of the topological basis
</ParamField>

<ParamField path="auto_tag" type="bool" default="True">
  Automatically annotate the circuit with topological information to improve parallelization.
</ParamField>

<ResponseField name="return" type="cirq.Circuit">
  The optimized circuit broken into native CZ gate set of {CZ, PhXZ}, where each moment is as parallel as possible.
</ResponseField>

<Note>
Extra topological information (similarity) can be used by tagging each gate with the topological basis groups that it belongs to:
```python
circuit.append(cirq.H(qubits[0]).with_tags(1, 2, 3, 4))
```
This represents that the gate is part of topological basis groups 1, 2, 3, and 4.
</Note>

#### Example

```python
import cirq
from bloqade.cirq_utils import parallelize

q = cirq.LineQubit.range(3)
circuit = cirq.Circuit(
    cirq.H(q[0]),
    cirq.H(q[1]),
    cirq.CX(q[0], q[1]),
    cirq.CX(q[1], q[2])
)

optimized = parallelize(circuit)
print(optimized)
```

### transpile

```python
from bloqade.cirq_utils import transpile

transpiled = transpile(circuit)
```

Transpile a circuit to a native CZ gate set of {CZ, PhXZ}.

<ParamField path="circuit" type="cirq.Circuit" required>
  The circuit to transpile.
</ParamField>

<ResponseField name="return" type="cirq.Circuit">
  The transpiled circuit using only CZ and PhasedXZ gates.
</ResponseField>

### auto_similarity

```python
from bloqade.cirq_utils import auto_similarity

tagged_circuit, weights = auto_similarity(
    circuit,
    weight_1q=1.0,
    weight_2q=2.0
)
```

Automatically tag the circuit with topological basis group labels.

<ParamField path="circuit" type="cirq.Circuit" required>
  The circuit to analyze. Should be CZ + PhasedXZGate, otherwise no annotation will occur.
</ParamField>

<ParamField path="weight_1q" type="float" required>
  The weight to assign to single-qubit gates.
</ParamField>

<ParamField path="weight_2q" type="float" required>
  The weight to assign to two-qubit gates.
</ParamField>

<ResponseField name="return" type="tuple[cirq.Circuit, dict[Hashable, float]]">
  - `[0]`: The circuit with each gate annotated with topological similarity tags
  - `[1]`: A dictionary mapping each tag to its weight
</ResponseField>

### moment_similarity

```python
from bloqade.cirq_utils import moment_similarity

tagged_circuit, weights = moment_similarity(circuit, weight=1.0)
```

Associate every gate in each moment with a similarity group.

<ParamField path="circuit" type="cirq.Circuit" required>
  The circuit to be analyzed.
</ParamField>

<ParamField path="weight" type="float" required>
  The weight to assign to each block of gates.
</ParamField>

<ResponseField name="return" type="tuple[cirq.Circuit, dict[Hashable, float]]">
  - `[0]`: The circuit with each gate annotated with topological similarity tags
  - `[1]`: A dictionary mapping each tag to its weight
</ResponseField>

### block_similarity

```python
from bloqade.cirq_utils import block_similarity

tagged_circuit, weights = block_similarity(circuit, weight=1.0, block_id=0)
```

Associate every gate in a circuit with a similarity group.

<ParamField path="circuit" type="cirq.Circuit" required>
  The circuit to be analyzed.
</ParamField>

<ParamField path="weight" type="float" required>
  The weight to assign to each block of gates.
</ParamField>

<ParamField path="block_id" type="int" required>
  The block identifier.
</ParamField>

<ResponseField name="return" type="tuple[cirq.Circuit, dict[Hashable, float]]">
  - `[0]`: The circuit with each gate annotated with topological similarity tags
  - `[1]`: A dictionary mapping each tag to its weight
</ResponseField>

### remove_tags

```python
from bloqade.cirq_utils import remove_tags

clean_circuit = remove_tags(circuit)
```

<ParamField path="circuit" type="cirq.Circuit" required>
  The circuit to remove tags from.

<ResponseField name="return" type="cirq.Circuit">
  The circuit with all tags removed.
