## Overview

Squin provides a comprehensive set of quantum gates for circuit construction. All gates can be applied to single qubits or broadcast to multiple qubits.

## Single-Qubit Pauli Gates


## Universal Gates

### u3

Apply a U3 gate with three rotation parameters:

```python
squin.u3(theta, phi, lam, qubit)
```

The U3 gate is defined as: U3(θ, φ, λ) = Rz(φ)Ry(θ)Rz(λ)

<ParamField path="theta" type="float" required>
  Rotation around Y axis (radians)
</ParamField>

<ParamField path="phi" type="float" required>
  Global phase shift component (radians)
</ParamField>

<ParamField path="lam" type="float" required>
  Z rotations in decomposition (radians)
</ParamField>

<ParamField path="qubit" type="Qubit" required>
  Target qubit
</ParamField>

### phased_xz

Apply a PhasedXZ gate:

```python
squin.phased_xz(x_rad, z_rad, axis_phase_rad, qubit)
```

Equivalent to Rz(-axis_phase) Rx(x) Rz(axis_phase + z) with angles in radians.

<ParamField path="x_rad" type="float" required>
  X rotation angle in radians
</ParamField>

<ParamField path="z_rad" type="float" required>
  Z rotation angle in radians
</ParamField>

<ParamField path="axis_phase_rad" type="float" required>
  Axis phase (XY plane) in radians
</ParamField>

<ParamField path="qubit" type="Qubit" required>
  Target qubit
</ParamField>

## Broadcast Operations

All gates can also be applied to multiple qubits at once using the broadcast API:

```python
from bloqade import squin
from kirin.dialects import ilist

@squin.kernel
def example():
    qubits = squin.qalloc(5)
    
    # Apply Hadamard to all qubits
    squin.broadcast.h(qubits)
    
    # Apply RX rotation to all qubits
    squin.broadcast.rx(1.57, qubits)
```

For broadcast operations, the function signatures are identical except the `qubit` parameter becomes `qubits` with type `ilist.IList[Qubit, Any]`.

## Example Circuit

```python
from bloqade import squin

@squin.kernel
def bell_state():
    """Create a Bell state."""
    q = squin.qalloc(2)
    
    squin.h(q[0])
    squin.cx(q[0], q[1])
    
    return squin.measure(q)
```
