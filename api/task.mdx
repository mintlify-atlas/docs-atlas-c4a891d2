## BatchFuture

Protocol for future objects that can be awaited.

```python
class BatchFuture(abc.ABC, Generic[RetType])
```

### Methods

#### result

Returns the result of the future, blocking until it is available or the timeout expires.

```python
def result(self, timeout: float | None) -> list[RetType]
```

<ParamField path="timeout" type="float | None" required>
  Timeout in seconds. If None, wait indefinitely until the result is available.
</ParamField>

<ResponseField name="return" type="list[RetType]">
  The result of all executions in the batch.
</ResponseField>

**Example:**

```python
future = device.run_async(kernel=my_kernel, shots=10)

# Wait up to 30 seconds for results
results = future.result(timeout=30.0)
```

#### partial_result

Return all results that are available so far, or MISSING_RESULT for those that are not yet available.

```python
def partial_result() -> list[RetType | MISSING_RESULT]
```

<ResponseField name="return" type="list[RetType | MISSING_RESULT]">
  A list containing available results and MISSING_RESULT (None) for pending executions.
</ResponseField>

**Example:**

```python
future = device.run_async(kernel=my_kernel, shots=100)

# Check partial results while still running
partial = future.partial_result()
print(f"Completed: {sum(1 for r in partial if r is not None)}")
```

#### fetch

Fetches the result of the future that are currently available.

```python
def fetch() -> None
```

**Example:**

```python
future = device.run_async(kernel=my_kernel, shots=100)

# Update internal state with available results
future.fetch()
partial = future.partial_result()
```

#### cancel

Attempts to cancel the execution of the future.

```python
def cancel()
```

<Tip>
This method may raise `NotImplementedError` if cancellation is not supported by the specific implementation.
</Tip>

#### cancelled

Returns True if the future was cancelled, False otherwise.

```python
def cancelled() -> bool
```

<ResponseField name="return" type="bool">
  True if the future was successfully cancelled, False otherwise.
</ResponseField>

<Tip>
This method may raise `NotImplementedError` if cancellation is not supported by the specific implementation.
</Tip>

#### done

Checks if all executions in the batch are complete.

```python
def done() -> bool
```

<ResponseField name="return" type="bool">
  True if all executions are complete, False otherwise.
</ResponseField>

**Example:**

```python
future = device.run_async(kernel=my_kernel, shots=100)

# Poll until complete
while not future.done():
    time.sleep(1)
    print("Still running...")

results = future.result(timeout=None)
```

```python
@dataclass
class AbstractTask(abc.ABC, Generic[Params, RetType])
```

## AbstractRemoteTask

Base class for tasks generated by remote devices.

```python
@dataclass
class AbstractRemoteTask(AbstractTask[Params, RetType])
```

### Methods

#### run

Executes the kernel synchronously and returns the result.

```python
def run(self, *, shots: int = 1, timeout: float | None = None) -> list[RetType]
```

<ParamField path="shots" type="int" default="1">
  The number of times to run the kernel method.
</ParamField>

<ParamField path="timeout" type="float | None" default="None">
  Timeout in seconds. If None, wait indefinitely.
</ParamField>

<ResponseField name="return" type="list[RetType]">
  The result of all executions.
</ResponseField>

**Example:**

```python
task = device.task(kernel=my_kernel)
results = task.run(shots=10, timeout=30.0)
```

#### run_async

Executes the kernel asynchronously and returns a Future object.

```python
def run_async(self, *, shots: int = 1) -> BatchFuture[RetType]
```

<ParamField path="shots" type="int" default="1">
  The number of times to run the kernel method.
</ParamField>

<ResponseField name="return" type="BatchFuture[RetType]">
  A future object for tracking execution progress.
</ResponseField>

**Example:**

```python
task = device.task(kernel=my_kernel)
future = task.run_async(shots=100)
results = future.result(timeout=60.0)
```

## AbstractSimulatorTask

Base class for tasks generated by local simulators.

```python
@dataclass
class AbstractSimulatorTask(
    AbstractTask[Params, RetType], Generic[Params, RetType, StateType]
)
```

### Methods

#### run

Executes the kernel and returns the result.

```python
def run(self) -> RetType
```

<ResponseField name="return" type="RetType">
  The result of the kernel execution.
</ResponseField>

**Example:**

```python
task = simulator.task(kernel=my_kernel)
result = task.run()
```

### Properties

#### state

Returns the state of the simulator after running the task.

```python
@property
def state(self) -> StateType
```

<ResponseField name="return" type="StateType">
  The internal state of the simulator after execution.
</ResponseField>

**Example:**

```python
task = simulator.task(kernel=my_kernel)
result = task.run()
state = task.state
```
